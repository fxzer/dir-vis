## 两种方案
### master分支: 利用 NodeJS 脚本生成目录结构数据, 再渲染成图  
### pure-front分支: 利用 [showDirectoryPicker](https://developer.mozilla.org/en-US/docs/Web/API/Window/showDirectoryPicker) API 选择目录并递归生成目录结构数据
